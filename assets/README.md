# Assets Klasörü

Bu klasöre profil fotoğrafınızı ekleyin:

1. LinkedIn profilinizden veya başka bir kaynaktan profil fotoğrafınızı indirin
2. Fotoğrafı `profile-photo.jpg` olarak kaydedin (veya .png formatında `profile-photo.png` olarak)
3. HTML dosyasındaki `src="assets/profile-photo.jpg"` kısmını kullandığınız formata göre güncelleyin

**Önerilen özellikler:**
- Minimum 600x600 piksel (kaliteli görünüm için)
- JPG veya PNG formatı
- Kare format (1:1 oran)

Fotoğraf eklendikten sonra sitenizde otomatik olarak görünecektir.
